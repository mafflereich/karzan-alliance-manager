# Kazran Alliance Manager - 功能手冊

**Kazran Alliance Manager** 是一個專為遊戲聯盟設計的綜合管理平台，旨在協助管理者高效追蹤多個公會成員的遊戲進度、角色練度與專屬武器持有狀況。

## 1. 系統概述

本系統分為 **前台 (公會視角)** 與 **後台 (管理視角)** 兩大區塊。
- **前台**：提供直觀的視覺化圖表，讓成員與幹部快速檢視各公會的戰力分佈。
- **後台**：提供完整的資料庫管理功能，包含公會名單、角色服裝資料、帳號安全與系統備份。

## 2. 使用者權限體系

系統設有三種權限層級，確保資料安全與管理分工：

| 角色 | 功能權限 |
| :--- | :--- |
| **最高管理者 (Admin/Creator)** | 擁有系統最高權限。可存取所有公會前台、完整的後台管理功能（包含密碼修改、備份、進階工具）。 |
| **聯盟幹部 (Manager)** | 擁有全域檢視權限。可自由切換並查看所有公會的服裝練度表，方便進行跨公會戰力分析，但無法修改系統設定。 |
| **公會成員 (Guild User)** | 僅能登入並查看所屬公會的儀表板。無法切換至其他公會或查看敏感資料。 |

## 3. 核心功能介紹

### 3.1 登入與導航
*   **分級公會列表**：首頁依據公會戰力梯隊 (Tier) 進行分組顯示，介面清晰。
*   **安全驗證**：支援公會專屬密碼登入與管理員個人帳號登入。
*   **權限導向**：系統依據登入身分，自動導向至對應的權限頁面。

### 3.2 公會儀表板 (前台)
這是系統的核心介面，提供以下功能：
*   **成員名單管理**：顯示成員名稱、職位（會長、副會長、成員）與最後更新時間。
*   **視覺化練度表**：
    *   **矩陣視圖**：橫軸為角色服裝，縱軸為公會成員，一目了然。
    *   **練度色塊**：使用不同顏色（灰、藍、紫、橘）直觀區分練度等級 (+0 ~ +5)。
    *   **專武標示**：若成員持有該角色的專屬武器，會顯示特殊圖示。
*   **快速切換**：管理者與幹部可透過側邊欄快速在不同公會間切換。
*   **資料編輯**：授權人員可直接點擊表格進行成員練度資料的更新。

### 3.3 後台管理系統 (僅限管理者)

#### A. 公會管理
*   **公會維護**：新增、編輯或刪除公會資料。
*   **梯隊排序**：可設定公會的所屬梯隊與顯示順序，影響首頁的排列方式。
*   **成員管理**：可進入特定公會進行成員名單的增刪修補。

#### B. 服裝資料庫管理
*   **角色管理**：新增遊戲角色，並支援自定義排序。
*   **服裝管理**：
    *   為角色新增對應的服裝外觀。
    *   **新品標示**：可將特定服裝標記為 **NEW**，在列表中優先顯示。
    *   **拖曳排序**：支援直觀的拖曳方式調整角色與服裝的顯示順序。

#### C. 帳號安全管理
*   **單一密碼重置**：快速重置特定使用者的登入密碼，具備安全強度檢查。
*   **批次密碼更新**：
    *   支援匯出公會帳號名單。
    *   可透過 CSV 檔案批次上傳並更新多個帳號的密碼，適合定期維護使用。
    *   介面提供即時的處理進度與成功/失敗統計。

#### D. 系統備份與還原
*   **完整備份**：一鍵匯出系統所有資料（公會、成員、練度、設定）為備份檔。
*   **災難復原**：支援從備份檔還原系統資料，並設有防呆機制避免誤操作。

#### E. 進階工具
*   **自動化搬運**：整合外部工具，協助將舊有資料快速導入系統。
*   **資料清理**：自動偵測並移除重複或無效的成員資料，保持資料庫整潔。
*   **服裝表匯入**：支援從外部來源批次匯入服裝定義資料。

## 4. 系統邏輯特色

1.  **智慧排序**：
    *   公會依據「梯隊」與「自訂順序」排列。
    *   成員依據「職位高低」排列。
    *   服裝依據「新推出優先」與「自訂順序」排列。

2.  **安全性設計**：
    *   嚴格的權限隔離，確保一般成員無法觸及後台設定。
    *   敏感操作（如密碼修改）皆經過伺服器端驗證。

3.  **友善操作體驗**：
    *   全站移除干擾性彈窗，改用現代化的狀態提示訊息。
    *   支援響應式設計，方便在不同裝置上查看。
